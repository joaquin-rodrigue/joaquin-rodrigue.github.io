About Me

Portfolio

A Simple CMS

This website makes several features of a Content Management System in a simple manner using Node.js, Express, and MySQL.

I'm learning a lot about generating SQL dynamically, server-side code, working with cookies, building webpages, proper CSS, database structures, database management, and other general JavaScript while working with this website.

It's been fun designing simple but functional UI/UX, and learning how some more complex server features work beyond what is being taught in classes.

You can find the GitHub repo for this project here: https://github.com/joaquin-rodrigue/simple-cms

Buried

This is an old project I've recently brought up to speed with more recent projects of mine. This game made in Unity is a 2-dimensional platformer made from scratch in art and programming.

I got to work with both programming in Unity and developing art assets, and more recently I've updated other aspects of the project using modern Unity features. I also originally developed the project with AR integration via an Ultraleap hand tracking sensor, but unfortunately I have not had much access to it since and these features may be out of date now.

This project was really the first full game I made using Unity, I found not only designing the game but building the levels, art, adding in gameplay features, and generally every part of the process was enjoyable.

MonopolAR

This group project I worked on led to my team recreating Monopoly through a phone-based AR system. We emulated the gameplay from a phone, playable by using a Monopoly board to allow showing off a 3-dimensional monopoly simulation.

I worked mostly on developing the AR integration with this project, but also worked on other features of Monopoly such as the player's ability to get cards and interactions with board squares.

Some of the group I worked on this project with are good friends still to this day, and I found the challenge of trying to understand AR and the various hardware problems we encountered alongside it to be fun. The hardware problems were fairly tedious, but otherwise the project was great.
